# Anime Time

An app for browsing anime titles and tracking episode completion! 

## Technologies Used

-HTML 5 \
-CSS3 \
-JavaScript \
-Bootstrap 5.1.3
-Node.js, Express, Mongoose
-MongoDB
-Jikan MyAnimeList API (https://jikan.docs.apiary.io/#)
